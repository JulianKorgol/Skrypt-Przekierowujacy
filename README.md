# Skrypt-Przekierowujacy
Skrypt przekierowujący umożliwia po wrzuceniu na stronę przekierowanie użytkownika na wskazany wcześniej URL.

W linijce kodu "<meta http-equiv="refresh" content="0; https://www.stronajuliana.com.pl/" />" zamiast "https://www.stronajuliana.com.pl/" należy umieścić własny adres strony.
